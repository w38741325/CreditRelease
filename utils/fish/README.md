## Fish shell completions for Credit
This folder has basic Fish completions for `creditd`, `credit-wallet-cli`, and `credit-wallet-rpc`. To use them, put those files (or symlink them) inside `~/.config/fish/completions/` or wherever your Fish completion files are (see [https://fishshell.com/docs/current/completions.html#where-to-put-completions](https://fishshell.com/docs/current/completions.html#where-to-put-completions)).
